How it works
1. Select a champion (only A's work see "Broken" below)
2. Select a star level
3. Select items (effects don't work, only pure stat buffs, except for Blue Buff and Shojin)
4. Select augments (effects don't work, only pure stat buffs)
5. Select desired active traits :
    =>The selected champion must have the trait or have an emblem equipped to get the buff if the trait only applies to itself (e.g. challengers).
    =>Team wide buffs (e.g. bruiser) will work regardless.
6. Select number of enemies hit (multihit applies to the spell if it can hit multiple champions (e.g. Aphelios spell, not Akshan)
Enjoy.
